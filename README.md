# Password-Security-Checker
Password Security Checker es un script en Python que simula la descifración de contraseñas utilizando métodos aleatorios de fuerza bruta o inteligentes carácter por carácter. Analiza la intensidad de la contraseña a través de la entropía, combinaciones y tiempos estimados de descifrado. Herramienta educativa para aprender ciberseguridad.
